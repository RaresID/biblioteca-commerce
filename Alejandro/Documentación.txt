# Introducción al proyecto

El proyecto consiste en la creación de biblioteca enfocada a la venta de los libros
que mostrará los datos relacionados con:
- Libros
- Género Literario
- Editores
- Clientes

# Ideas para el proyecto

La primera idea consiste en organizar el código de manera clasificada:
- Modelos (para dividir los apartados anteriores en una base de datos)
- Esquemas (con el fin de establecer los atributos acorde al modelo)
- Fichero principal (unificar el Frontend y el Backend)
